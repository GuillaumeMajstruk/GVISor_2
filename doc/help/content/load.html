A l'ouverture du programme ou en cours d'utilisation:

Pour charger une GVI déjà commencée:

  1. Cliquer sur le bouton "Fichier" en haut à gauche de la page d'acceuil.
  2. Cliquer ensuite sur "Charger une GVI..."

Choisir un fichier de sauvegarde à charger:

Une fenêtre apparaît alors (Le répertoire d'enregistrement des GVI en cours) dans celle-çi se trouvent un ou plusieurs fichier(s) de sauvegarde de GVI (format .gtmp)
il est aussi possible que celui-çi soit vide si aucune GVI n'est en cours de réalisation.

Le nom du fichier contient les informations qui vont permettre de sélectionner la sauvegarde correcte
ils sont enregistrés au format suivant: SnMoteur_NumeroDeLancement_DateDeLaGvi(.gtmp).

Sélectionner un fichier dans la liste, puis cliquer sur "ouvrir"


Le fichier a été chargé avec succés:

Si le fichier a été correctement chargé, une fenêtre apparaît pour le spécifier en indiquant le nom du fichier chargé. Cliquer sur "ok" pour passer à l'étape suivante.

Si ce n'est pas le fichier voulu il suffit de retourner à la première étape de cette page pour charger le fichier voulu.


Le fichier n'a pas pu être chargé:

Si le fichier n'a pas été correctement chargé, cela peut être dû à plusieurs facteurs: se reporter à la section "Erreurs" de cette aide.

Essayez de charger à nouveau le fichier comme indiqué dans la première étape.


Ajouter un utilisateur et/ou continuer le chargement:

Une fois le chargement du fichier de sauvegarde effectué, les utilisateurs sont automatiquements ajoutés à la suite à partir du moment où le fichier chargé est modifié.

Les champs Numéro de lancement, SN moteur, Type moteur, Date, Client, Opérateurs, Utilisateur actuel sont remplis automatiquement.

Cliquer sur le bouton "Prendre en compte les informations du chantier" pour reprendre la GVI dans l'état où elle avait été laissée.


Continuer la GVI:

Le processus de chargement est alors arrivé à son terme, ainsi, il vous est possible de continuer la GVI dans l'état dans lequel elle était à l'enregistrement précédent.


En cas d'erreur lors du démarrage ou du fonctionnement du programme:
merci de se reporter à la section Gérer les problèmes d'utilisation.
